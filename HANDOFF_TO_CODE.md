## 1. 项目是什么

**survival-model** —— 一个**确定性**的行星 + 生命演化模拟。产物是两个可直接双击运行的 HTML:

- **`world.html`** —— 局部地点图,**逐分钟**模拟(气候+地质+生命)。**主文件,几乎所有逻辑都在这**(纯 JS,无构建)。
- **`planet.html`** —— 全球 2D 经纬网格视图(环流/海洋/气压/冰/板块)。
- **`design_overview.md`** —— 架构/守恒/模块/涌现/缺口的总规格。**改任何东西前先读它**,尤其末尾几节(根因一/三、谱系树、时序档位、geoLifeStep、比照真实进化的缺口)。

## 2. 四条铁律(绝不能破)

1. **属性=存量,动作=流量**。
2. **单一突变入口**。
3. **零随机** —— 所有事件由元素**跨阈确定性涌现**(张弛/relaxation-oscillation)。演化用 **adaptive dynamics**(性状沿适应度梯度爬)+ 阈值解锁创新,**不掷骰子**。允许"内生混沌"(多谱系竞争的非线性)当对称破缺,但**禁止外生 PRNG**。
4. **守恒 + 涌现**(没有一条结果是写死的)。

## 3. 工作流(每个特性都照此)

中文讲机制 → **python/node 数值验证**(守恒、张弛节律、涌现) → 落进 JS → 写进 `design_overview.md`。**先验证,再实现。**

## 4. 已建成(全程确定性、守恒、零随机)

完整生命演化弧:化学起源 → 自复制子 → 化能/光合代谢 → 原始细胞 → 大氧化(GOE) → 真核(内共生) → 多细胞 → 分化 → 有性 → 微生物食物网 → r/K → 矿化骨骼(寒武) → 神经 → 登陆(臭氧门) → 温血 → 互利共生 …… **18+ 可遗传性状**。
- **根因一**:平均场→**离散谱系**(`r.life=[]` 簇 + 全局 `phylo[]` 档案)+ **演化分支**(adaptive dynamics:歧化选择奇异点确定性分裂)→ 见 `updateLineages()`。
- **谱系树可视化**:`drawPhylo()` + overlay(按钮「🌳谱系树」),按地点着色、悬停详情卡(且悬停时暂停时间)。
- **时序档位**:速度按钮=时间单位(小时/天/月/年/10年/百年/千年/万年)。细档走 `stepMinute`,粗档走 `geoYear`+`geoLifeStep`(地质年生命步,弛豫积分,任意倍率稳定、量级对齐分钟级)。
