# 人体数据模型 v0.1 · 食物与水（推导文档）

配套代码：`BodyModel.cs`（纯 C#，无引擎依赖，可 xUnit 直接测）。

## 设计原则

- **存量-流量 + 反馈回路**：每个生理量是一个"存量"，被一组"流量"增减；流量依赖其他存量 → 自然产生关联。
- **症状而非数字**：底层用连续数值，给玩家的是离散状态（口渴/脱水/饥饿）。
- **机能输出**：不直接掉 HP，而是降低"体力上限/认知"等机能，机能再决定能做什么。

## 存量（Stocks）

| 量 | 字段 | 参考/初值 | 说明 |
|----|------|-----------|------|
| 体水 | `WaterMl` | 42000 mL（≈60%体重） | 脱水严重度的来源 |
| 糖原 | `GlycogenKcal` | 1800 上限 | 快能量缓冲，≈静息一天 |
| 脂肪 | `FatG` | 12000 g | 慢能量库，9 kcal/g，<3000g 瘦死 |
| 胃内容 | `_gKcal/_gWaterMl/_gProteinG/_gSaltG` | 0 | 吃下去先进胃，按排空速率释放 |
| 待排溶质 | `_pendingSoluteMOsm` | 0 | 强制排水的来源 |

## 流量（Flows）

**进水**：喝水 + 食物含水 + **代谢水**（`消耗kcal × 0.13 mL`）
**出水**：不感蒸发（≈40 mL/h，随活动↑）+ 出汗（随温度/活动）+ **尿** + 粪
**进能量**：胃排空→吸收（受 `MaxAbsorbKcalPerHour` 上限）→ 先充糖原、溢出转脂肪
**出能量**：`BMR × 活动倍率`；不足时先抽糖原后抽脂肪

**尿（关键）**：肾脏排溶质有浓缩上限（`MaxUrineConc≈1200 mOsm/L`）。
`必需尿量 = 待排溶质 / 1200 × 1000 mL`；再取与基础尿量的较大值；体水超参考时额外利尿。

## 两条把食物和水"焊死"的耦合

1. **溶质负荷强制排水**：蛋白质（`×4 mOsm/g`）和盐（`×35 mOsm/g`）必须由尿带水排出。
   → 干吃高蛋白/高盐而水不够 = **越吃越脱水**（现实中的"兔肉饥饿"）。
2. **代谢水回补 vs 活动耗水**：烧食物产生少量水；但活动/高温又通过出汗、呼吸多耗水。

## 验证（默认常量，Python 同构跑）

- **断水·22°C静息**：~17h 口渴(2%)，~3.5天 危险(10%)，~5天 致命(15%) ✅ 合现实
- **断食·喝够水**：糖原 1 天耗尽，脂肪线性下降，**第 40 天**到脂肪临界死亡 ✅
- **耦合验证（同样每天 400mL 水，22°C）**：

  | 吃什么 | 6天后脱水 | 死亡 |
  |--------|-----------|------|
  | 牛肉干 200g/天 | 18.2% | **第5天** |
  | 不吃只喝水 | 16.2% | 第6天 |
  | 椰肉 200g/天 | 15.2% | 第6天 |

  吃牛肉干的人比吃椰肉的人**早死一天**——纯由食物对水平衡的影响涌现。✅

## 待标定（重要）

- ⚠️ **出汗系数偏高**：`SweatPerDegPerHour=60` 导致 35°C 静息也按重度出汗算（~11L/天）。
  建议降到 ~15–25，或等接入体温系统后用"热负荷"驱动出汗，替换这条经验式。
- 断水致死在静息温和环境下偏慢（5天），高温会快很多；可按你的气候设定再调 `Insensible/Sweat`。
- 其余常量（BMR、糖原容量、脂肪临界、溶质系数）量级合理，可按难度微调。

## 怎么测（xUnit 思路）

```csharp
[Fact]
public void 断水五天内致命() {
    var b = new HumanBody();
    for (int h = 0; h < 24*6 && !b.IsDead; h++) b.Step(1, EnvInput.Rest(22));
    Assert.True(b.IsDead && b.DehydrationPct >= 15);
}
[Fact]
public void 高蛋白比高水食物更易脱水() {
    var dry = new HumanBody(); var wet = new HumanBody();
    for (int d = 0; d < 6; d++) for (int h = 0; h < 24; h++) {
        if (h==8){ dry.Eat(FoodItem.Jerky(200)); wet.Eat(FoodItem.Coconut(200)); }
        if (h%6==0){ dry.Drink(100); wet.Drink(100); }
        dry.Step(1, EnvInput.Rest(22)); wet.Step(1, EnvInput.Rest(22));
    }
    Assert.True(dry.DehydrationPct > wet.DehydrationPct); // 牛肉干更脱水
}
```

## 下一步可接（保持同一套 stock-flow）

- **电解质**（钠/钾）：出汗丢盐 → 低钠抽筋/意识障碍；与"喝水但不补盐反而危险"耦合。
- **体温系统**：用热平衡驱动出汗/颤抖，替换当前出汗经验式；颤抖额外烧糖原。
- **疾病/伤口**：发热抬高代谢与失水；失血直接减体水/血容量。
- **营养素细分**：维生素/必需氨基酸 → 慢变量缺乏症（接 Card Survival 那种深度）。
