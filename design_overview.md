## 1. 设计哲学（四条纪律）

整套世界建立在一套统一的数据驱动框架上（实现见 `world.html` 内的 JS 模型）：

- **属性 = 存量，动作 = 流量**：每个量是一个数（存量），只被一组"动作"（流量）增减。
- **单一增减入口**：运行期改属性只走 `World.Add/Set`（唯一入口，便于审计、可回放、不会两段代码偷偷打架）。
- **无随机**：任何"事件"都是各元素交互到临界的**必然**，不掷骰子。复杂行为靠**张弛振荡**（攒过阈值→突然释放→复位）涌现——雨的滞回、闪电的电荷击穿、盐的析出、火山的岩浆压力、板块的应力，都是同一套引擎，只是时间尺度不同。
- **守恒 + 涌现**：每条流量都是"转移"不是凭空增减；行为不是写死的，而是少数局部规则互相咬合后跑出来的。

> 关键判断：**温度即热能的代理**，所以没有把整套重写成焦耳引擎；而是补上真正断掉的反馈（潜热、反照率），让能量回路闭合。同理，地质用"张弛 + 守恒"而非随机。

**行星参数（魔法数的推导根）**：一组变化极慢的行星基本量当常量（`PLANET = {质量, 半径, 自转周期, 轨道天数, 恒星光通量, radio, metallicity}`，默认地球），由它**推导**出原先写死的常数——重力 `g=GM/R²`、大气标高 `H=RT/(M_air·g)`（替代气压式里的 8000）、科里奥利 `Ω=2π/自转`（替代 7.29e-5）、气温直减率 `∝g`、日照倍率 `∝恒星光度/轨道距离²`，以及读出量（逃逸速度、平均密度、每度公里）。默认地球值精确复现既有魔法数（g≈9.8、H≈8000、Ω≈7.27e-5、直减率 6.5），故标定不变；改 `PLANET` 即换一颗星球（高 g→大气更薄、直减率更陡；快自转→科里奥利更强）。

**周期表 + 核合成演化（`PERIODIC`）**：整张周期表分层——33 种有真实循环的单独追踪，其余按化学/起源并成 ~16 组（聚合丰度 + 起源标签）。宇宙丰度由**核合成**定：大爆炸(H/He)、恒星聚变(C→Fe 峰，丰)、超新星/中子星(>Fe 重元素，稀)。行星的重元素含量由 **`metallicity`（星系化学演化代，太阳=1）** 缩放（超新星产物 ∝ Z^1.3）。**接根**：`radioFuel ∝ metallicity` → `coreActivity` → 磁场/火山/大气存亡；重金属矿(Cu/Sn/Pb/Ag/Au/Hg/U/Th) ∝ `metallicity`。于是**第一代贫金属行星(Z→0)：无 U/Th→冷核无发电机、无矿、大气被剥（死星）；晚代富金属(Z>1)：强放射热、富矿、长寿**——元素起源（宇宙演化代）决定行星命运。

## 2. 时间架构（三个尺度的心跳）

唯一权威量是 `GameClock.TotalMinutes`（单调递增，日历全靠 `/` 和 `%` 推导，永不丢时间）。三级心跳：

| 心跳 | 周期 | 驱动什么 |
|------|------|----------|
| `MinuteTick` | 每游戏分钟 | 天气/水文/局部全部按分钟推进（`WorldMap.Minute` → 各 Region 事件 + 路由 + 扩散） |
| `DayTick` | 每游戏天 | 全球 `Planet.StepDay`（慢变量）+ 把全球边界推给局部 `PushBoundary` |
| 每游戏年 | 365 游戏天 | `Geology.StepYear`（火山/地震/侵蚀/固结/板块漂移——地质慢变量） |

现实 1 秒 = 游戏 1 分钟（`RealSecondsPerGameMinute=1`）；快进/旅行用 `Advance(分钟)` 逐分钟补足。

**按变化速率分摊算力**（性能）：量按其真实变化尺度选更新周期，慢量不每分钟重算。
- 每分钟：温度快层、水汽/云/降水/径流、风、地下水（必须高频）。
- 每天：环影/环光（只随赤纬变，省掉 24 步几何循环的逐分钟重算——实测一年 52 万次→366 次）。日/月食每步只算一次（日+月各一），HUD/光照/月亮/潜热各处**复用同一缓存值**，不再各自重算。
- 每小时（`CHEM=60`）：矿物风化、**土壤发育/剥蚀**（地质级慢量，×CHEM 补频率）。
- 每年（`geoYear`）：火山/地震/侵蚀/固结/板块、放射成因热、米兰科维奇、大气总量。
> 原则：缓存只随某尺度变的量、把慢过程挪到低频心跳，物理结果不变、算力大降。

## 3. 四本守恒账（+ 第五本岩石账）

像水/矿物一样，能量与碳也被当作守恒量追踪——这是抓 bug 与产生自稳反馈的支柱。

| 账本 | 包含 | 源 / 汇 | 实现/校验 |
|------|------|---------|-----------|
| **水** | 土壤+**地下水**+积雪+冰川+低空水汽+云+湖+在途 | 无（全在库间转移：蒸发/降水/径流/扩散/相变/深渗-基流） | 地下水=慢库（土壤深渗补给→旱季基流出泉）；全球冰↔海洋体积守恒→**冰川性海平面**升降 |
| **矿物** | 23 元素的 溶解+沉积 | 源=岩石风化（近无限慢源）；搬运守恒 | `WaterChemistry`；蒸发只走纯水→咸海/盐滩涌现 |
| **碳** | 全球 `GlobalCO2` | 源=火山脱气；汇=**陆地硅酸盐风化** + **海底风化**（碳酸化海底玄武岩） | 地质恒温器(Walker)：暖→风化快→吸碳→变冷，自稳。**双汇**让水世界(无陆)也有(较弱)恒温，陆比例=连续旋钮 |
| **能量** | 吸收太阳 vs 长波散失 | 太阳入、辐射出 | `WorldMap.EnergyBalance()`；平衡时净≈0，扰动后回平衡 |
| **岩石**（隐式） | 基岩 `Elev` + 沉积物 `Sediment` | 源=深部岩浆（火山）；侵蚀↔固结↔俯冲重熔 | 闭合循环：火成→风化→沉积→固结→俯冲→熔融→火成 |

## 4. 模块清单（按层；每个=作用 + 关键涌现）

### 天体 / 全球（慢变量）
- **YearCycle**：赤纬+朔望月——季节、日长、月相全由"第几天"推导。
- **米兰科维奇轨道周期 → 冰期**：倾角(`TILT`)/偏心率(`ecc`)/岁差(`EQDAY`)在 `geoYear`（深时）里极慢确定性振荡（周期压缩到地质快进可见，比例守 41:100:23 kyr）→ 调制季节日照（`milankFlux`）→ 与**冰反照率+冰川**咬合 → **冰期—间冰期锯齿张弛**自己涌现。纯天文、零随机——正是"事件由天体演化到阈值涌现"。
- **大气总量演化 `atmMass`**：总大气压成慢存量——火山**脱气**(源) vs **逃逸**(太阳风剥∝1/磁屏蔽 + 俯冲埋藏本底汇 + 热逃逸∝1/重力)。地球（屏蔽1）脱气≈汇→稳；**火星式（弱场+弱火山）→逃逸>>源→剥到近光**（接磁场死亡链）。随构造活跃度小幅高低。
- **Planet**（2D 经纬网格，按天）：三圈环流×科里奥利→信风/西风/极地东风；海温/盐度/西岸寒流；季风；**气压场**（四气压带+热力高低压）+**锋面/风暴轴**；**冰反照率**（极地放大）；**板块构造**（对流→板块速度→汇聚/离散）。给局部当边界。
- **Tides / 卫星**：`MOONS` 是**卫星列表**（可多颗，各自 {质量,距离,周期}），由质量/距离推导每颗的潮汐力(∝M/d³)与视亮度(∝半径²/d²)。夜光、潮汐都**对列表求和**，每颗用自己的周期。单颗=月+日叠加的规则大小潮；多颗→各周期+太阳**拍频**，潮差不规则、连珠时超级大潮，夜里多月各自盈亏升落——全是免费涌现，加卫星只动夜光与潮汐两处。卫星质量放大（如火星大小）：潮汐∝M/d³、夜光∝M^⅔/d²、视盘∝R 都照样 scale（数米巨潮、亮如四满月）；但质量再大质心移出主星→实为**双行星**、且伴星本身是独立世界（模型只模拟单地表）——属适用域边界（见 7.5）。
- **日月食 `inc`（轨道倾角）**：每颗卫星带轨道倾角。**确定性涌现**——只有新月/满月恰好碰上卫星过**交点**（黄道面，黄纬β≈0）时才成食；朔望月与交点月不同周期→食季每~半年漂移一次，故食**稀少**（验证：现月球日食~1.4h/年、月食~4.3h/年；零倾角则每次朔望都食→证明倾角是稀有性来源）。**日食**=卫星遮日，削减日照→白昼骤暗骤冷+暗盘咬日（全食正中全黑、星辰显现）；**月食**=满月入地影→夜里月变暗泛红（血月）。卫星增大→食更频更长易全食。只动日照与夜光两处，纯局部天象（`world.html`）。
- **行星环 `RING`**（土星式赤道盘 {内缘,外缘,不透明度,反照}）：两个**确定性**效应，都挂在已有赤纬上、关环=现状。①**环影**：环在**冬半球**投下阴影、削减日照（几何积分得削减比例∝不透明度，分点环侧对日→归零，δ翻号→换半球）→ 冬季深寒（局部 `world.html` 直接乘 (1-环影) 到吸收太阳，由年均锚自然封底；全球 `planet.html` 给目标温有界致冷 `RING_K`，验证中纬冬季加深 ~15℃，夏季不变）。②**环光**：被照亮的环把光反射到夜半球（∝|sinδ|，夏半球看到亮环面更多），接月光量纲加进夜间照度。③视觉：`world.html` 天空里一道横跨的环弧（含卡西尼暗缝，夜里更醒目）。只动日照+夜光两处。

### 大气 / 气候
- **SunCycle / 恒星**：`SUNS` 是**恒星列表**（可多颗=双星系统，各 {光通量,视日长,相位}），`sunF`=对每颗高度×光通量**求和** → 光照与加热。单颗=现有（标定基准）；两颗视日长不同→**白昼/日照拍频**（对齐=双日高温+真夜，错开=长昼少夜），平均日照更高→更暖。馈入光/温度/蒸发/全球环流自动响应。月亮见下 `Tides/卫星`。
- **Albedo**：反射率调制吸收太阳（相对基准，裸地不回归）；雪冰高反照→**冰反照率正反馈**。
- **Climate**：双层热惯性（地温慢层 + 气温快层）；海拔降基准；CO₂温室偏差；潜热汇。→ 午后最热、夏至后最热（嵌套滞后）。
- **水汽温室反馈**（Clausius-Clapeyron）：饱和水汽压随温（每16℃翻倍≈4.4%/℃，锚定15℃→默认不变）→暖→水汽多→温室强→更暖。**正反馈**，把 CO₂/轨道/火山强迫放大~1.67×（Python 验证有界、不失控、与冰反照率合成雪球/失控温室阈值）。
- **Humidity**：相对湿度与气温反相（午后干、黎明潮）。
- **Cloud + 云反照率反馈**：云挡日照（短波致冷）+ 夜保温（长波增温）→压平昼夜温差。**暖→蒸发→云→反射阳光↑→致冷**（负反馈），与水汽增温**打擂台**→净气候敏感度被夹在合理区间。
- **Storm**：电荷张弛→闪电(IC/CG~2.7:1)；**冬无雷/旱雷涌现**；闪电驱动固氮；风暴降温。
- **Wind**：盛行风 + 白天海风/谷风 + 夜下坡风 + 阵风 → 风寒体感、加速蒸发、吹尘。

### 水文 / 冰冻
- **WaterCycle**：土壤/积雪/湖；蒸发→成云→雨/雪→融雪→径流；潜热（蒸发致冷/凝结放热/融化平台）；冻土阻下渗。
- **地下水/含水层**：土壤近饱和→深渗补给慢库（`GW`）；含水层慢基流→径流→**旱季泉流**维持河水。守恒（水在土壤↔含水层↔径流间转移）。
- **土壤发育（pedogenesis）**：`soilDepth` 风化造土（化学+物理）÷ 径流×坡度剥土 → 平衡土深（陡湿坡薄、平缓厚）；土深调制持水容量 `soilCap`（锚定→默认不变）。为日后生命铺垫。
- **Waves**：浪高向风²弛豫→**成长(duration)+余涌(swell)**；海冰镜面；拍岸浪侵蚀海岸。
- **Dust**：松尘↔悬浮尘；风超阈起尘×干燥→**尘暴**遮日致冷→顺风沉积**黄土**（无植被锁土的招牌）。
- **Ice**：**冻融风化**(0°C 附近最强)→岩粉喂尘；**冰川**(常年雪压实/慢水库/刨蚀)；冻土；海冰。**冰川性海平面**：全球冰量(冰川+雪)↔海洋体积守恒→冰期冰多→**海平面降**（冰期—海退连锁，配合米兰科维奇）。
- **热盐环流（MOC，全球）**：高纬冷+咸→密→下沉驱动；**向极输热**（暖高纬）。淡水输入（冰融脉冲）→盐度降→密度降→下沉停→**MOC 崩塌**→极地骤冷（双稳/滞回：撤淡水也未必自启）。`planet.html`「融冰淡水」开关可现场触发停摆。

### 岩石圈 / 地质
- **WaterChemistry**：碳酸驱动风化→**33 元素**溶出 + **吸 CO₂** + **剥蚀岩石**；蒸发浓缩析盐壳。元素分组：造岩/盐(Na/Ca/Mg/K/SiO₂/Cl/SO₄/HCO₃…)、金属矿(Fe/Cu/Sn/Pb/Ag/Au/Al/Mn/Ti/Ni/Cr)、生命微量(I/Zn/Mo/Co/Se/P)、火药(硝/S/C)、**放射性(U/Th)**、毒/助熔(Hg/B/F)。岩性决定产谱(长英质富U/Th/Al、镁铁质富Ni/Cr/Co/Ti、黏土富Al…)。
- **金属成矿（蒸发盐壳之外的三路真实成矿）**：矿床判定=沉积 `dep>0.05`。除"蒸发浓缩"（盐/硝/硼砂在干涸盆地结壳）外，补三条金属成矿：①**热液脉**——火山喷发时岩浆热液在火山口+邻区沉淀 Cu/Pb/Zn/Ag/Au/Sn/Mo/Hg 硫化物套（火山区成铜铅锌金矿带）；②**红土残积**——湿热高地深风化把可溶组分淋走、不溶氧化物 Fe/Al/Mn/Ti/Ni/Cr 残留富集（铝土/红土镍/铁帽，湿热专属，冷干不形成）；③**砂矿**——重稳矿物 Au/Sn/Ti 随侵蚀搬运在下游低能段按密度富集。金属矿物=稳定氧化物/硫化物，**沉淀后锁定不回溶**（`LOCKED`，盐仍按溶解度遇水化掉）。→ 矿床出现在地质上该出现处：火山口(热液)、湿热高地(红土)、河谷盆地(砂矿+蒸发)，而非全挤在盐盆。
- **海洋元素闭环（堵"洋盆死胡同"）**：溶解质随径流汇入海后，原本无去路、地质尺度无限漂高。补上**海洋沉积埋藏**——海区把**超出海水本底**(`SEAREF`)的溶解质缓慢埋入海底自生矿物→`subPool`(俯冲缓冲池)→**火山喷发随岩浆返还**→再风化，闭合"幔–壳溶解载荷"回路（验证：海水成分收敛到本底附近稳态、且海洋+池总量守恒，不再漂移）。一举闭合 Na/Cl/Ca/Si/SO₄/**P** 等的海洋收支。**氮非生物闭合**：硝酸盐随俯冲入火山高温**热解放 N₂ 回大气**（不返还为硝），与闪电固氮平衡——这是地质过程而非生物反硝化，故无生命世界也成立。（真正需要生物的汇——生物硅、有机碳埋藏、生物反硝化——按设计留给"生命"层。）
- **Terrain / Lithology**：地形静态参数 + 6 种岩性（各自矿物产率）。
- **Geology**（按年，张弛）：火山（攒压→喷发，自重自调节）、地震、侵蚀、**固结成沉积岩**（成分定岩型）、**板块构造**（活动度调火山地震、俯冲重熔、大陆漂移）。
- **磁场（地磁发电机）**：B ∝ **核活动(内部热)×自转**（偶极→磁极强）。耦合：**大气屏蔽/逃逸**（强场保住大气；弱场→太阳风剥 CO₂/N₂）、**地表辐射场**（磁极 funnel/薄气/高海拔/弱场→辐射高，生存危害）、**极光**（高磁纬+场 channel）。核活动**与火山同根**——`magmaP ∝ coreActivity`：小行星内部冷却→**发电机+火山一起熄→大气被剥→冷干辐照死世界（火星）**。
  且 `coreActivity` 不再是自由参数，而是**放射成因热推导**：`= 放射燃料(U/Th/K含量,按半衰期衰变) / 散热(∝1/R)`——地球(满放射料、散热慢)≈1 数十亿年；小行星(散热快)起步就低、早死；贫铀行星生来偏冷。这把"行星为何有内部热/为何会死"接到了**放射性元素**上。另：**氡(Rn)** 由 U 衰变产、低通风洞穴积聚→辐射危害（接洞穴大气 + 辐射场）；地表辐射 += U/Th 岩石本底。

### 嵌套
- **WorldMap**：地点图（节点=Region，边=径流/扩散/路线）；每分钟三步（气候→路由→扩散）；持有 `Planet`+经纬度，每天把全球态（气候基准/海温/盛行风/过境天气）推给各地点。
- **世界类型 `WORLDTYPE`**：`continent`(大陆，默认枢纽山谷) / `ocean`(全海洋，唯一陆地=热点火山岛+礁/潟湖/开阔洋/深海)。切换只换**地点集+路线+火山口+起点**，物理规则不变。`planet.html` 也有「全海洋」开关(清空大陆掩膜)。**水世界验证**：清陆→40°N 季节温差 30→14°C(海惯性熨平)、季风/西岸寒流/雨影/尘暴(皆需陆海对比)消失；碳靠海底风化兜底→CO₂ 稳在偏高值而非失控(若无此汇会温室跑飞)。

### 生命前体 / 摇篮（非生物地基，已就绪）
- **史前有机化学（蛋白质，生命之前）**：两存量 `org`(有机单体≈氨基酸) + `prot`(多肽/蛋白质)。**合成**(能量=闪电+喷口ΔG × 原料 CO₂×固氮N × 还原环境)→**缩合聚合**(催化=黏土/矿物表面 × 浓缩=干涸/吸附，双分子∝org²)↔**水解**(湿相断键)→**缓慢降解**(无氧极慢，所以才能攒)。守恒(合成扣无机C、降解归还)，确定性，**干湿张弛**(潮汐/季节湖：干相聚合↔湿相水解→净累积)。验证：**潮汐黏土池 & 深海喷口**累积出蛋白(史实两路，几十年到可观量)；开阔洋(无催化)→只有单体不聚合、有氧世界→氧化降解压制。这是化学演化、还**没有复制/选择**——是第一缕生命的原料库。
- **🦠 第一缕生命：自我复制分子（RNA 世界式自催化）**——存量 `repl` + 可遗传性状 `rTopt`(耐温最适)。**自催化复制**(∝`repl` × Monod 底物[吃有机汤 org+prot] × 温度适应度)——造更多 repl 的速率正比于已有 repl，即指数增长，生命的签名。**确定性起源**：当有机汤在摇篮处累积**过阈** `rIgnite` 且条件合适 → 复制**点燃**(初代 `rTopt`=当地温)，否则不点燃——起源是环境到阈值的确定性事件，不靠运气(验证：温带潮池/喷口点燃、开阔洋贫汤永不点燃)。**确定性演化(环境选择)**：`rTopt` 按 `+adapt×(当地温−rTopt)` **确定性爬向当地温度**(定量遗传学的适应度梯度结果)——同一复制子在冷处演化耐冷、暖处耐暖，零随机(验证：Topt→15/→3)。**张弛**：暴食有机汤→汤崩→种群受限，由持续的非生物合成续命。守恒(食→repl→死亡碎屑→食)。**伏笔**：异养吃光有限的原始汤 → 对"自己造食物"的选择压 → 下一步**化能自养**阈值解锁。
- **海底热液喷口**：深海/近火山海区(`hvent`)喷热(局部暖点)+ 还原流体 **H₂/H₂S**(蛇纹石化, `redu`) + 金属硫化物烟囱(喂金属循环)。生命起源最主流的摇篮环境。
- **氧化还原 / 化学自由能梯度** `freeEnergy`：还原剂(`redu`，喷口)与海水氧化剂(CO₂/SO₄)**失衡 = 可用化学能**——第一代谢的能源。喷口处 ΔG 最高。
- **前生命"摇篮"就绪度** `cradleIndex`(0–1，显示用、不反馈)：双通道取 max——**喷口通道**(水×暖×自由能×硫化物催化×CHNOPS) / **地表通道**(水×温和×闪电能×干湿循环×黏土催化×CHNOPS)。验证：深海喷口~50%、湿地黏土池~23% 点亮；开阔洋/冻极/旱漠/洞穴≈0。两类经典起源环境自然涌现。
- **就绪清单**(全部非生物已满足)：液态水 ✓、还原性无氧 CO₂-N₂ 大气 ✓、闪电能(Miller-Urey) ✓、CHNOPS ✓、催化金属(Fe/Ni/Co/Mo/Zn/Mn/Se) ✓、矿物模板(黏土/硫化物) ✓、干湿循环(潮汐/蒸发湖) ✓、稳定气候+闭合养分循环 ✓、热液喷口+ΔG ✓。→ **模型已具备生命起源的非生物基础**；下一步"点燃生命"是生命层的事。

## 5. 全球 ↔ 局部 嵌套

局部地点图是**球面某 (纬,经) 的高分辨率局部**。全球 `Planet`（粗网格，按天）算出大尺度态，通过 `PushBoundary` 每天喂给局部：

- **陆地气候基准** ← 全球纬度年均温（锚定 40°N=既有标定；移动纬度即换气候带）。
- **海岸海温** ← 全球 SST（含西岸寒流）。
- **盛行风** ← 全球纬向风（退役了静态 `oro` 雨影）。
- **过境天气** ← 全球气压/锋面（`SynopticWetness`：低压/锋面→湿、高压→干）。
- **大陆漂移** → 局部纬度随地质年缓变 → 整套气候/生物带改写。

## 7.5 适用域 / 边界（重要：别误用）

本模型从骨子里是一个 **「固体地表 + 薄流体包层（大气/海）」的类地天体模型**：地形/海拔、水循环、岩石循环、风化、沉积、板块、地点图、生存——**一切以基岩表面为参照**，能量回路以**入射太阳 vs 长波**为主导、内热为微小的放射成因热。

- **域内（能模拟，多为换配置）**：岩石行星、卫星、**水世界**（唯一陆地=火山岛，海底风化兜底碳恒温）。可调旋钮：行星参数(质量/半径/自转/轨道/金属丰度…)、纬度、世界类型、卫星/恒星列表(多卫星/双星)。
- **域外（撞穿根本假设，非调参可补）**：**气态/冰巨星（木星、海王星）、恒星**。原因：① **没有固体表面**——没有"地面"可站、"海平面"可参照，被模拟的那个地表世界整体失效；② **能量内热主导**（木星向外辐射 ≈1.7× 吸收阳光，来自原始/KH 收缩热），模型的主导项里没有；③ 环流是**深对流条带急流**而非类地薄壳 Hadley；④ 成分是 **H₂/He 包层**，无硅酸盐 → 无碳恒温器。要模拟它们需另起一个**竖直压力分层的流体包层模型**（压力坐标而非海拔坐标、内热驱动），那是另一类产品，不是本模型的开关。

> 验证旁注：把木星参数喂进**推导根**公式，`g=GM/R²`(25.9，实测24.8)、`H=RT/Mg`(H₂/He 23.8km，实测~27)、`Ω=2π/T`(2.4×地球)、`1/d²`(阳光仅1/27) **都给对**——推导层是天体无关的，边界出在它**下游**那个"地表世界"。
