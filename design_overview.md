## 1. 设计哲学（四条纪律）

整套世界建立在一套统一的数据驱动框架上（实现见 `world.html` 内的 JS 模型）：

- **属性 = 存量，动作 = 流量**：每个量是一个数（存量），只被一组"动作"（流量）增减。
- **单一增减入口**：运行期改属性只走 `World.Add/Set`（唯一入口，便于审计、可回放、不会两段代码偷偷打架）。
- **无随机**：任何"事件"都是各元素交互到临界的**必然**，不掷骰子。复杂行为靠**张弛振荡**（攒过阈值→突然释放→复位）涌现——雨的滞回、闪电的电荷击穿、盐的析出、火山的岩浆压力、板块的应力，都是同一套引擎，只是时间尺度不同。
- **守恒 + 涌现**：每条流量都是"转移"不是凭空增减；行为不是写死的，而是少数局部规则互相咬合后跑出来的。

> 关键判断：**温度即热能的代理**，所以没有把整套重写成焦耳引擎；而是补上真正断掉的反馈（潜热、反照率），让能量回路闭合。同理，地质用"张弛 + 守恒"而非随机。

**行星参数（魔法数的推导根）**：一组变化极慢的行星基本量当常量（`PLANET = {质量, 半径, 自转周期, 轨道天数, 恒星光通量, radio, metallicity}`，默认地球），由它**推导**出原先写死的常数——重力 `g=GM/R²`、大气标高 `H=RT/(M_air·g)`（替代气压式里的 8000）、科里奥利 `Ω=2π/自转`（替代 7.29e-5）、气温直减率 `∝g`、日照倍率 `∝恒星光度/轨道距离²`，以及读出量（逃逸速度、平均密度、每度公里）。默认地球值精确复现既有魔法数（g≈9.8、H≈8000、Ω≈7.27e-5、直减率 6.5），故标定不变；改 `PLANET` 即换一颗星球（高 g→大气更薄、直减率更陡；快自转→科里奥利更强）。

**周期表 + 核合成演化（`PERIODIC`）**：整张周期表分层——33 种有真实循环的单独追踪，其余按化学/起源并成 ~16 组（聚合丰度 + 起源标签）。宇宙丰度由**核合成**定：大爆炸(H/He)、恒星聚变(C→Fe 峰，丰)、超新星/中子星(>Fe 重元素，稀)。行星的重元素含量由 **`metallicity`（星系化学演化代，太阳=1）** 缩放（超新星产物 ∝ Z^1.3）。**接根**：`radioFuel ∝ metallicity` → `coreActivity` → 磁场/火山/大气存亡；重金属矿(Cu/Sn/Pb/Ag/Au/Hg/U/Th) ∝ `metallicity`。于是**第一代贫金属行星(Z→0)：无 U/Th→冷核无发电机、无矿、大气被剥（死星）；晚代富金属(Z>1)：强放射热、富矿、长寿**——元素起源（宇宙演化代）决定行星命运。

## 2. 时间架构（三个尺度的心跳）

唯一权威量是 `GameClock.TotalMinutes`（单调递增，日历全靠 `/` 和 `%` 推导，永不丢时间）。三级心跳：

| 心跳 | 周期 | 驱动什么 |
|------|------|----------|
| `MinuteTick` | 每游戏分钟 | 天气/水文/局部全部按分钟推进（`WorldMap.Minute` → 各 Region 事件 + 路由 + 扩散） |
| `DayTick` | 每游戏天 | 全球 `Planet.StepDay`（慢变量）+ 把全球边界推给局部 `PushBoundary` |
| 每游戏年 | 365 游戏天 | `Geology.StepYear`（火山/地震/侵蚀/固结/板块漂移——地质慢变量） |

现实 1 秒 = 游戏 1 分钟（`RealSecondsPerGameMinute=1`）；快进/旅行用 `Advance(分钟)` 逐分钟补足。

**按变化速率分摊算力**（性能）：量按其真实变化尺度选更新周期，慢量不每分钟重算。
- 每分钟：温度快层、水汽/云/降水/径流、风、地下水（必须高频）。
- 每天：环影/环光（只随赤纬变，省掉 24 步几何循环的逐分钟重算——实测一年 52 万次→366 次）。日/月食每步只算一次（日+月各一），HUD/光照/月亮/潜热各处**复用同一缓存值**，不再各自重算。
- 每小时（`CHEM=60`）：矿物风化、**土壤发育/剥蚀**（地质级慢量，×CHEM 补频率）。
- 每年（`geoYear`）：火山/地震/侵蚀/固结/板块、放射成因热、米兰科维奇、大气总量。
> 原则：缓存只随某尺度变的量、把慢过程挪到低频心跳，物理结果不变、算力大降。

## 3. 四本守恒账（+ 第五本岩石账）

像水/矿物一样，能量与碳也被当作守恒量追踪——这是抓 bug 与产生自稳反馈的支柱。

| 账本 | 包含 | 源 / 汇 | 实现/校验 |
|------|------|---------|-----------|
| **水** | 土壤+**地下水**+积雪+冰川+低空水汽+云+湖+在途 | 无（全在库间转移：蒸发/降水/径流/扩散/相变/深渗-基流） | 地下水=慢库（土壤深渗补给→旱季基流出泉）；全球冰↔海洋体积守恒→**冰川性海平面**升降 |
| **矿物** | 23 元素的 溶解+沉积 | 源=岩石风化（近无限慢源）；搬运守恒 | `WaterChemistry`；蒸发只走纯水→咸海/盐滩涌现 |
| **碳** | 全球 `GlobalCO2` | 源=火山脱气；汇=**陆地硅酸盐风化** + **海底风化**（碳酸化海底玄武岩） | 地质恒温器(Walker)：暖→风化快→吸碳→变冷，自稳。**双汇**让水世界(无陆)也有(较弱)恒温，陆比例=连续旋钮 |
| **能量** | 吸收太阳 vs 长波散失 | 太阳入、辐射出 | `WorldMap.EnergyBalance()`；平衡时净≈0，扰动后回平衡 |
| **岩石**（隐式） | 基岩 `Elev` + 沉积物 `Sediment` | 源=深部岩浆（火山）；侵蚀↔固结↔俯冲重熔 | 闭合循环：火成→风化→沉积→固结→俯冲→熔融→火成 |

## 4. 模块清单（按层；每个=作用 + 关键涌现）

### 天体 / 全球（慢变量）
- **YearCycle**：赤纬+朔望月——季节、日长、月相全由"第几天"推导。
- **米兰科维奇轨道周期 → 冰期**：倾角(`TILT`)/偏心率(`ecc`)/岁差(`EQDAY`)在 `geoYear`（深时）里极慢确定性振荡（周期压缩到地质快进可见，比例守 41:100:23 kyr）→ 调制季节日照（`milankFlux`）→ 与**冰反照率+冰川**咬合 → **冰期—间冰期锯齿张弛**自己涌现。纯天文、零随机——正是"事件由天体演化到阈值涌现"。
- **大气总量演化 `atmMass`**：总大气压成慢存量——火山**脱气**(源) vs **逃逸**(太阳风剥∝1/磁屏蔽 + 俯冲埋藏本底汇 + 热逃逸∝1/重力)。地球（屏蔽1）脱气≈汇→稳；**火星式（弱场+弱火山）→逃逸>>源→剥到近光**（接磁场死亡链）。随构造活跃度小幅高低。
- **Planet**（2D 经纬网格，按天）：三圈环流×科里奥利→信风/西风/极地东风；海温/盐度/西岸寒流；季风；**气压场**（四气压带+热力高低压）+**锋面/风暴轴**；**冰反照率**（极地放大）；**板块构造**（对流→板块速度→汇聚/离散）。给局部当边界。
- **Tides / 卫星**：`MOONS` 是**卫星列表**（可多颗，各自 {质量,距离,周期}），由质量/距离推导每颗的潮汐力(∝M/d³)与视亮度(∝半径²/d²)。夜光、潮汐都**对列表求和**，每颗用自己的周期。单颗=月+日叠加的规则大小潮；多颗→各周期+太阳**拍频**，潮差不规则、连珠时超级大潮，夜里多月各自盈亏升落——全是免费涌现，加卫星只动夜光与潮汐两处。卫星质量放大（如火星大小）：潮汐∝M/d³、夜光∝M^⅔/d²、视盘∝R 都照样 scale（数米巨潮、亮如四满月）；但质量再大质心移出主星→实为**双行星**、且伴星本身是独立世界（模型只模拟单地表）——属适用域边界（见 7.5）。
- **日月食 `inc`（轨道倾角）**：每颗卫星带轨道倾角。**确定性涌现**——只有新月/满月恰好碰上卫星过**交点**（黄道面，黄纬β≈0）时才成食；朔望月与交点月不同周期→食季每~半年漂移一次，故食**稀少**（验证：现月球日食~1.4h/年、月食~4.3h/年；零倾角则每次朔望都食→证明倾角是稀有性来源）。**日食**=卫星遮日，削减日照→白昼骤暗骤冷+暗盘咬日（全食正中全黑、星辰显现）；**月食**=满月入地影→夜里月变暗泛红（血月）。卫星增大→食更频更长易全食。只动日照与夜光两处，纯局部天象（`world.html`）。
- **行星环 `RING`**（土星式赤道盘 {内缘,外缘,不透明度,反照}）：两个**确定性**效应，都挂在已有赤纬上、关环=现状。①**环影**：环在**冬半球**投下阴影、削减日照（几何积分得削减比例∝不透明度，分点环侧对日→归零，δ翻号→换半球）→ 冬季深寒（局部 `world.html` 直接乘 (1-环影) 到吸收太阳，由年均锚自然封底；全球 `planet.html` 给目标温有界致冷 `RING_K`，验证中纬冬季加深 ~15℃，夏季不变）。②**环光**：被照亮的环把光反射到夜半球（∝|sinδ|，夏半球看到亮环面更多），接月光量纲加进夜间照度。③视觉：`world.html` 天空里一道横跨的环弧（含卡西尼暗缝，夜里更醒目）。只动日照+夜光两处。

### 大气 / 气候
- **SunCycle / 恒星**：`SUNS` 是**恒星列表**（可多颗=双星系统，各 {光通量,视日长,相位}），`sunF`=对每颗高度×光通量**求和** → 光照与加热。单颗=现有（标定基准）；两颗视日长不同→**白昼/日照拍频**（对齐=双日高温+真夜，错开=长昼少夜），平均日照更高→更暖。馈入光/温度/蒸发/全球环流自动响应。月亮见下 `Tides/卫星`。
- **Albedo**：反射率调制吸收太阳（相对基准，裸地不回归）；雪冰高反照→**冰反照率正反馈**。
- **Climate**：双层热惯性（地温慢层 + 气温快层）；海拔降基准；CO₂温室偏差；潜热汇。→ 午后最热、夏至后最热（嵌套滞后）。
- **水汽温室反馈**（Clausius-Clapeyron）：饱和水汽压随温（每16℃翻倍≈4.4%/℃，锚定15℃→默认不变）→暖→水汽多→温室强→更暖。**正反馈**，把 CO₂/轨道/火山强迫放大~1.67×（Python 验证有界、不失控、与冰反照率合成雪球/失控温室阈值）。
- **Humidity**：相对湿度与气温反相（午后干、黎明潮）。
- **Cloud + 云反照率反馈**：云挡日照（短波致冷）+ 夜保温（长波增温）→压平昼夜温差。**暖→蒸发→云→反射阳光↑→致冷**（负反馈），与水汽增温**打擂台**→净气候敏感度被夹在合理区间。
- **Storm**：电荷张弛→闪电(IC/CG~2.7:1)；**冬无雷/旱雷涌现**；闪电驱动固氮；风暴降温。
- **Wind**：盛行风 + 白天海风/谷风 + 夜下坡风 + 阵风 → 风寒体感、加速蒸发、吹尘。

### 水文 / 冰冻
- **WaterCycle**：土壤/积雪/湖；蒸发→成云→雨/雪→融雪→径流；潜热（蒸发致冷/凝结放热/融化平台）；冻土阻下渗。
- **地下水/含水层**：土壤近饱和→深渗补给慢库（`GW`）；含水层慢基流→径流→**旱季泉流**维持河水。守恒（水在土壤↔含水层↔径流间转移）。
- **土壤发育（pedogenesis）**：`soilDepth` 风化造土（化学+物理）÷ 径流×坡度剥土 → 平衡土深（陡湿坡薄、平缓厚）；土深调制持水容量 `soilCap`（锚定→默认不变）。为日后生命铺垫。
- **Waves**：浪高向风²弛豫→**成长(duration)+余涌(swell)**；海冰镜面；拍岸浪侵蚀海岸。
- **Dust**：松尘↔悬浮尘；风超阈起尘×干燥→**尘暴**遮日致冷→顺风沉积**黄土**（无植被锁土的招牌）。
- **Ice**：**冻融风化**(0°C 附近最强)→岩粉喂尘；**冰川**(常年雪压实/慢水库/刨蚀)；冻土；海冰。**冰川性海平面**：全球冰量(冰川+雪)↔海洋体积守恒→冰期冰多→**海平面降**（冰期—海退连锁，配合米兰科维奇）。
- **热盐环流（MOC，全球）**：高纬冷+咸→密→下沉驱动；**向极输热**（暖高纬）。淡水输入（冰融脉冲）→盐度降→密度降→下沉停→**MOC 崩塌**→极地骤冷（双稳/滞回：撤淡水也未必自启）。`planet.html`「融冰淡水」开关可现场触发停摆。

### 岩石圈 / 地质
- **WaterChemistry**：碳酸驱动风化→**33 元素**溶出 + **吸 CO₂** + **剥蚀岩石**；蒸发浓缩析盐壳。元素分组：造岩/盐(Na/Ca/Mg/K/SiO₂/Cl/SO₄/HCO₃…)、金属矿(Fe/Cu/Sn/Pb/Ag/Au/Al/Mn/Ti/Ni/Cr)、生命微量(I/Zn/Mo/Co/Se/P)、火药(硝/S/C)、**放射性(U/Th)**、毒/助熔(Hg/B/F)。岩性决定产谱(长英质富U/Th/Al、镁铁质富Ni/Cr/Co/Ti、黏土富Al…)。
- **金属成矿（蒸发盐壳之外的三路真实成矿）**：矿床判定=沉积 `dep>0.05`。除"蒸发浓缩"（盐/硝/硼砂在干涸盆地结壳）外，补三条金属成矿：①**热液脉**——火山喷发时岩浆热液在火山口+邻区沉淀 Cu/Pb/Zn/Ag/Au/Sn/Mo/Hg 硫化物套（火山区成铜铅锌金矿带）；②**红土残积**——湿热高地深风化把可溶组分淋走、不溶氧化物 Fe/Al/Mn/Ti/Ni/Cr 残留富集（铝土/红土镍/铁帽，湿热专属，冷干不形成）；③**砂矿**——重稳矿物 Au/Sn/Ti 随侵蚀搬运在下游低能段按密度富集。金属矿物=稳定氧化物/硫化物，**沉淀后锁定不回溶**（`LOCKED`，盐仍按溶解度遇水化掉）。→ 矿床出现在地质上该出现处：火山口(热液)、湿热高地(红土)、河谷盆地(砂矿+蒸发)，而非全挤在盐盆。
- **海洋元素闭环（堵"洋盆死胡同"）**：溶解质随径流汇入海后，原本无去路、地质尺度无限漂高。补上**海洋沉积埋藏**——海区把**超出海水本底**(`SEAREF`)的溶解质缓慢埋入海底自生矿物→`subPool`(俯冲缓冲池)→**火山喷发随岩浆返还**→再风化，闭合"幔–壳溶解载荷"回路（验证：海水成分收敛到本底附近稳态、且海洋+池总量守恒，不再漂移）。一举闭合 Na/Cl/Ca/Si/SO₄/**P** 等的海洋收支。**氮非生物闭合**：硝酸盐随俯冲入火山高温**热解放 N₂ 回大气**（不返还为硝），与闪电固氮平衡——这是地质过程而非生物反硝化，故无生命世界也成立。（真正需要生物的汇——生物硅、有机碳埋藏、生物反硝化——按设计留给"生命"层。）
- **Terrain / Lithology**：地形静态参数 + 6 种岩性（各自矿物产率）。
- **Geology**（按年，张弛）：火山（攒压→喷发，自重自调节）、地震、侵蚀、**固结成沉积岩**（成分定岩型）、**板块构造**（活动度调火山地震、俯冲重熔、大陆漂移）。

### 生命前体 / 摇篮（非生物地基，已就绪）
- **史前有机化学（蛋白质，生命之前）**：两存量 `org`(有机单体≈氨基酸) + `prot`(多肽/蛋白质)。**合成**(能量=闪电+喷口ΔG × 原料 CO₂×固氮N × 还原环境)→**缩合聚合**(催化=黏土/矿物表面 × 浓缩=干涸/吸附，双分子∝org²)↔**水解**(湿相断键)→**缓慢降解**(无氧极慢，所以才能攒)。守恒(合成扣无机C、降解归还)，确定性，**干湿张弛**(潮汐/季节湖：干相聚合↔湿相水解→净累积)。验证：**潮汐黏土池 & 深海喷口**累积出蛋白(史实两路，几十年到可观量)；开阔洋(无催化)→只有单体不聚合、有氧世界→氧化降解压制。这是化学演化、还**没有复制/选择**——是第一缕生命的原料库。
- **🦠 第一缕生命：自我复制分子（RNA 世界式自催化）**——存量 `repl` + 可遗传性状 `rTopt`(耐温最适)。**自催化复制**(∝`repl` × Monod 底物[吃有机汤 org+prot] × 温度适应度)——造更多 repl 的速率正比于已有 repl，即指数增长，生命的签名。**确定性起源**：当有机汤在摇篮处累积**过阈** `rIgnite` 且条件合适 → 复制**点燃**(初代 `rTopt`=当地温)，否则不点燃——起源是环境到阈值的确定性事件，不靠运气(验证：温带潮池/喷口点燃、开阔洋贫汤永不点燃)。**确定性演化(环境选择)**：`rTopt` 按 `+adapt×(当地温−rTopt)` **确定性爬向当地温度**(定量遗传学的适应度梯度结果)——同一复制子在冷处演化耐冷、暖处耐暖，零随机(验证：Topt→15/→3)。**张弛**：暴食有机汤→汤崩→种群受限，由持续的非生物合成续命。守恒(食→repl→死亡碎屑→食)。**伏笔**：异养吃光有限的原始汤 → 对"自己造食物"的选择压 → **化能自养**阈值解锁（下一条）。
- **能量预算 / 维持代谢（显式耗能）**：生命是一本能量收支账——**净 = 收入（三代谢）− 维持代谢 − 死亡**。**维持代谢** `rMaintK`=活着就持续烧的能量（基础代谢 + 膜维持 + 主动移动并入），随温**温和**变化（冷省能/暖费能，但适应者各温皆可活，非陡 Q10 假死）。**收入 < 维持 → 饿死**（消耗自身生物量，不只是停长）→ 边际生境养不活生命。**维持代谢 = 呼吸**：烧有机物→**放 CO₂**（光合的逆反应，补全碳/氧循环另一半；O₂ 消耗并入全局氧收支）。为体型/温血/食物链能量传递打底（每营养级先付维持）。
- **化能自养（代谢革新，阈值解锁）**：第二个可遗传性状 `rAuto`(异养0↔化能自养1)，按"异养(吃汤) vs 自养(还原剂+CO₂)"两策略**人均适应度确定性漂移**。自养用喷口**还原剂(H₂)氧化 + 固定 CO₂** 自产有机物，受**资源供给限制**(还原剂/CO₂ 通量→有界，非指数爆炸)。**涌现**：异养复制子吃稀原始汤→喷口处自养适应度反超→`rAuto`→高→生命**挣脱原始汤天花板**(验证:生物量 ~0.8→~38)、稳靠 ΔG+CO₂ 续命；无 ΔG 处自养永不划算、停在异养低位。**固碳 = 第一台生物碳泵**(压低本地 CO₂，接回碳循环)。守恒(增殖≤底物×产率，死亡归还碎屑)。同一套确定性选择——汤足→吃白食、汤枯+有能量梯度→自产，环境定策略。
- **原始细胞（膜→个体化，选择单元升级）**：脂质 `lip`(有机合成副产物≈脂肪酸)浓度过 **CMC 阈值**→**自发组装膜泡**(张弛)。第三个可遗传性状 `rMemb`(膜化 0↔1)：膜带来**浓缩→增殖↑** + **保护→死亡↓**(代价=耗脂质+代谢负担)。膜化因**更能打**被选择——脂质过 CMC 处 `rMemb`→1 = **第一批原始细胞**；脂质稀处无膜、只有裸分子。**意义**：选择单元从"自由分子"升级为"有膜的个体"，是通往**可交互的具体生命个体**的桥(将来从种群-场实例化个体)。验证：潮池/喷口成细胞(膜100%)、开阔洋贫汤无膜≈0；喷口最富(细胞+兼性自养+高生物量)。确定性、阈值涌现、有界、守恒。
> 至此生命有三个性状在被环境确定性选择：**耐温 `rTopt`、代谢型 `rAuto`(异养↔化能自养)、膜化 `rMemb`**——同一套"适应度梯度爬坡 + 阈值解锁创新"，零随机。喷口=最富摇篮(原始细胞+化能自养+高生物量)。
- **光合作用 + 生命扩散（第三代谢，通往大氧化）**：第四性状 `rPhoto`(自养里光合 vs 化能)。光合用**阳光**(日均光 `litAvg` EMA，避昼夜闪烁误判) + 固 CO₂ + **放 O₂**。**生命扩散**（升级版）：① **邻接扩散 + 河流顺流平流**（被径流带向下游，∝runoff）——复制子从摇篮**殖民**到阳光表层；② **适应度过滤**：迁入者按目的地温度适应度成活，错配者死亡→有机碎屑（守恒）→ **温度屏障自动涌现**、且防 **gene-swamping**（源不再淹没汇）→ 各地保住本地适应、沿环境梯度**分异/物种形成**（验证：冷源迁入暖汇→汇仍本地适应 21.7℃，盲混则被拖到 13℃）。性状按存活生物量加权混合；③ **移动力 `rMove`(主动移动/趋性，行为萌芽)**：耗能(死亡↑)但能沿**生境质量 `lifeQ`(人均净增长)梯度定向迁往更优邻区**。按"可利用的梯度"被选择——**时变环境**(季节/昼夜/资源涨落)梯度持续→移动力被选上(验证 amp>0→rMove~100%)，恒定均质→纯成本→归零(amp=0→0%)。这是**趋性=行为的起点**，也是将来"会动、可交互的具体生物"的源头。生命的性状已增至**七个**：耐温 `rTopt`/代谢型 `rAuto`/光合 `rPhoto`/膜化 `rMemb`/移动力 `rMove`/**好氧度 `rAero`**/**盐度耐受 `rSalt`**
- **好氧度 `rAero`（O₂：废物→毒→资源，补全大氧化连锁）**：产氧光合的 O₂ 本是**废物**、对厌氧生命是**毒**（"氧灾变"）。`rAero` 让生命**先耐毒**（O₂ 致死 `(1+3·redox·(1-rAero))`，好氧者免疫）**再利用**（好氧呼吸产能↑，`w×(1+aerBoost·rAero·O₂)`），代价=好氧机器维护。**富氧→被选上**（产能+免毒 > 成本）、**缺氧→纯成本→归零**（厌氧者退守缺氧避难所）。验证：缺氧 rAero→0、富氧→100%。
- **盐度耐受 `rSalt`**：适应度=温度匹配×**盐度匹配**(`exp(-((盐度−rSalt)/宽)²)`)，`rSalt` 追当地盐度 → **海洋种/淡水种/嗜盐种**分异；扩散按盐度过滤 → **海淡屏障**（淡水种漂入海洋成活~0）。骑已有盐度系统。
- **捕食 + 体型 → 微生物食物网（仍单细胞阶段）**：① **体型 `rSize`**（第八性状）：大→增殖慢（权衡）+ 维持更省（Kleiber, 表面积/体积）+ **难被吃**（防御）；**体型选择 = r/K 选择**：`rSize += sizeAdaptK·(捕食防御 + (1−2·growthOpp))`，`growthOpp` 由 `lifeQ`(人均净增长)定 → **稳定/拥挤(已饱和, lifeQ≈0)→大个高效 K 策略；扰动频繁/在恢复占空地(lifeQ 高)→小个快繁 r 策略**；捕食压另推向大(防御)。验证：稳定→大(K)、高频扰动→小(r)、捕食→大。**r/K 谱由环境扰动频率确定性涌现**。② **消费者池 `cons`**（+耐温 `consTopt`）：吃**活的生产者** `repl`，按营养效率 `predYield`(~0.22) 转化 → **能量金字塔**（消费者 ≪ 生产者）；**捕食-猎物张弛**（生产者多→消费者暴发→过牧→崩→恢复）；消费者自身有维持/死亡/耐温适应/扩散；未同化+死亡→碎屑、呼吸→CO₂（守恒）。验证：能量金字塔 + 体型随捕食压振荡。**铺垫内共生**（吞噬=engulf，吞而不化→共生→真核细胞，下一条）。
- **内共生 → 真核细胞 `rEuk`（单细胞大跳跃）**：吞噬宿主（够大 `rSize`+能动 `rMove`）在**富氧**处吞下好氧菌**却不消化、留作线粒体**（吞蓝藻→叶绿体）→ 真核细胞。`rEuk` 选择 = `富氧(o2f) × 吞噬能力(大+能动) − 复杂度成本`：**只在大氧化之后（有 O₂ 供线粒体）+ 宿主能吞噬时涌现**（史实正确；缺氧或太小/不动→留原核）。真核收益：**线粒体→好氧能量暴涨**（`aerB += euBoost·rEuk·o2f`，验证 ×3）+ **养大体型半价**（能量支撑复杂度→突破原核的能量/体型天花板）。验证：富氧+大能动→真核(能量×3)、缺氧/小不动→原核。接 GOE+捕食+体型+移动力，**通往多细胞**。
- **多细胞 `rMulti`**：真核细胞抱团分工 → **大到单细胞捕食者吞不动（捕食免疫）**。选择 = `前提真核(rEuk) × 捕食压 − 成本`；成本=搭身体更慢长。**真核 + 捕食压 → 多细胞涌现**（被捕食率×0.04），无真核或无捕食 → 留单细胞（验证）。这是又一个重大转变，**通往复杂生物/可交互的具体个体**（仍在水里）。
- **细胞分化 `rDiff`（germ–soma + 分工）**：前提多细胞 × (稳定竞争 + 捕食) → 团块变成**有分工的生物体**：部分细胞专司繁殖（生殖系），其余成体细胞（摄食/防御/结构）。收益=专门化高效 + 防护组织；成本=只生殖系繁殖→繁殖产出降。稳定竞争/高捕食处分化、扰动/单细胞处留松散团块。**这才让多细胞从"团块"变"生物体"**，铺向身体计划/器官。
- **有性生殖 `rSex`（重组）**：前提真核（减数分裂）× **环境多变(慢性失配)** → 重组**加速适应**（红皇后；提升环境追踪型性状的适应率）。成本=**两倍成本**（繁殖慢）。多变/失配环境→有性、稳定环境→克隆省成本、原核→无。验证：稳定竞争→分化、多变→有性、缺前提→都不涌现。
> 生命性状已达 **12 个**：耐温/盐度/好氧度·代谢型/光合·体型·移动力·膜化·真核·多细胞·分化·有性（+消费者池）。整条演化弧（前生命化学→复制→代谢→细胞→光合→大氧化→食物网→真核→多细胞→分化→有性）全程确定性、守恒、阈值/选择涌现、零随机。
- **生物碳泵 / 有机碳埋藏（生命成为行星力量·盖娅耦合）**：死亡生物量一部分**埋藏**（**缺氧海底保存好** `(1−redox)`，充氧则腐解返 CO₂）→ **化石碳 `fossilC` + 本地煤 `dep[17]`（可采资源）+ 抽走 globalCO2**（温和，Walker 恒温器可吸收 → 长期**降温**，接气候/冰期）；`geoYear` 里**化石碳出露慢氧化返 CO₂**（闭合长期有机碳循环：埋藏锁碳 ↔ 风化释放）。这正是一直**留给生命层**的接管缺口——**生命现在反过来调控 CO₂/气候/矿产**（O₂ 侧由光合→大氧化承接）。验证：缺氧高产→埋藏成煤+抽 CO₂、充氧→不埋，有界。**至此生命与行星双向耦合闭环**。；那里**无原始汤、无喷口 ΔG**，只剩阳光 → 确定性选出**光合**(光是远比 ΔG 庞大、且到处都有的能源 → 生物量铺满阳光层)。O₂ 先被**还原剂/亚铁吃掉 → 条带状铁矿(BIF)**(接铁循环)。验证：暗喷口=化能、阳光层=光合 100%+放 O₂+成 BIF、暗富汤=异养——三选一+扩散确定性涌现。**伏笔**：O₂ 的全球累积(填满氧汇后)→ **大氧化事件**(下一条)。
- **大氧化事件（全局 O₂ 收支，延迟跃升）**：全局 `globalO2` + **还原缓冲库 `globalRed`**(海洋 Fe²⁺+火山还原气+还原地壳)。光合产氧先被**火山还原气(补给率)** + **氧化全局还原库(→条带铁矿 BIF)** 吃掉 → 长期压在 ≈0(前大氧化期、BIF 沉积)；当还原库耗尽**且产氧 > 还原物补给率** → O₂ **阈值式跃升 = GOE**(张弛：缓冲耗尽的相变)。`o2base` 耦合 `globalO2` → 大气把 O₂ **扩散到各地**；厌氧生命在富氧处受压(死亡↑→退守缺氧避难所)；净有机埋藏(放氧的另一面)抽 CO₂ → **降温**(接气候/可触发冰期)。验证：强光合→延迟后跃升+大量 BIF；弱光合(产<补)→永远缺氧、无 BIF。修好了"O₂ 瞬间冲顶"。缺氧期长短由缓冲库大小调。

### 生命演化后段（十大续作 ①–⑩，全程确定性/守恒/阈值-选择涌现、零随机）
- **① 大灭绝与辐射**：死亡项加**失配灭绝**项 `(1+extinctK·(1−fit))`——环境突变→`fit` 骤降→**大批死亡(灭绝脉冲)**；幸存者在腾空的生态位上**适应辐射**加速（`adaptMul` 含 `radK·lifeQ` 项，空位+净能量富余→适应加速）。**张弛**：扰动→灭绝→空位→快速辐射填充。无随机灭绝，全由环境跨阈触发。
- **② 营养级加深 `pred2`（食肉者→三级金字塔）**：消费者够密（`cons>预阈`）→**二级消费者/食肉者** `pred2` 起源，吃 `cons`，按 `predYield` 转化。能量金字塔加到**三层**（生产者≫食植者≫食肉者）；预-猎张弛叠一层。死亡/未同化→碎屑、呼吸→CO₂（守恒）。
- **③ 病毒与寄生 `parasite`（红皇后）**：宿主够密→寄生体起源；**传播 ∝ 宿主密度**、杀宿主→碎屑；**有性宿主抗性更强**（`resist=1−0.5·rSex`）。回头**驱动有性**：`rSex` 选择项加寄生压（红皇后——为甩开寄生而维持重组）。无宿主→载量衰退。
- **④ 生物闭合 N/Si 循环**：生命足够多（`repl>阈`）→**生物固氮**（补低氮、施肥 `dis[18]`）+**反硝化**（缺氧→硝酸盐转 N₂，闭氮，堵洋盆死胡同）；**生物硅**（取 SiO₂→埋藏，闭硅）。这接管了早先**特意留空给生命**的那几个生物地球化学汇。
- **⑤ 寒武爆发：矿化骨骼 `rShell`**：前提多细胞 × 捕食压 × **矿物可得度**（Ca `dis[1]`+碳酸盐 `dis[7]`）→**建矿化外壳/骨骼**防御（被捕食率 ×`(1−0.85·rShell)`）；代价=长得慢+耗钙。建壳取钙→死后→**石灰岩沉积** `dep[1]`（接岩石账）。军备竞赛（壳 vs ②③的捕食/寄生）=寒武爆发的引擎。
- **⑥ 感觉与神经 `rNeuro`**：前提多细胞 ×（移动力+捕食压）→**神经/感官**：**觅食增益**（`+neuroForage·rMove`）+**主动避敌**（被捕食率 ×`(1−0.6·rNeuro)`）；代价=神经能耗（维持↑）。行为从⑥的"趋性"升级为"定向觅食/逃避"。
- **⑦ 登陆套装 `rDry`+`rUV`+臭氧门**：`fit` 折入**干燥匹配** `exp(−((dry−rDry)/宽)²)` × **UV 存活** `(1−uvHarm·uvExp·(1−rUV))`，其中 `uvExp=(非海)×天空×(1−臭氧)`、**臭氧=clamp(globalO2/8)**（O₂→臭氧→屏蔽 UV）。水生不受影响（海:dry≈0、uvExp=0）。登陆 = **rDry+rUV 适应 × 需先有臭氧层**双门（验证：未适应+无臭氧 fit≈0.003 挡住；适应+有臭氧 fit≈0.92 放行）；扩散成活也按干燥度过滤 → **海陆屏障**。
- **⑧ 陆生植物改造陆地**：陆地光合生物（`repl×rDry×rPhoto`）→**根系把风化加速到~3×** → 多**抽 CO₂**（降温）+ 多**造土** + 多**释矿**。生命再次成为行星力量（继生物碳泵之后，这次改造**陆地**风化/气候）。
- **⑨ 温血 `rEndo`（体温调节）**：拓宽耐温窗 `tWidth=rWidth·(1+2.5·rEndo)`（偏离最适仍能活），代价=**维持代谢翻倍**（产热）。前提多细胞 × **冷胁迫**驱动（`coldStress=clamp((12−Tair)/24)`）——**只在寒冷/高纬划算**，暖处不选（验证：偏离最适25℃ fit 0.04→0.77、耗能+50%、暖处 rEndo 留 0）。
- **⑩ 个体实例化 `creatureOf(r)`**：读当地**性状分布**→确定性合成"此地主导个体"的**表型**：营养级（生产者/食生产者/顶级捕食者，由 repl/cons/pred2 谁主导定）、体制（裸复制子→原核→真核→多细胞→组织分化体）、体型/运动/防御（壳）/感官（神经）/栖息（海/陆/水缘）/温血。**命名 + 行为**一句话（如"陆生温血·大型组织分化体·顶级捕食者·有神经感官·追猎游动猎物"），挂 HUD。这是从"种群-场"到**可交互具体个体**的桥（实例化的第一步，交互待续）。

**验证**:布局算法隔离验证(森林多根/DFS叶序/灭绝节点坐标全对);drawPhylo+updatePhyloPanels 用 stub canvas/DOM 跑通无报错(节点标签/统计/面板内容正确)。canvas/HTML 终极验证需浏览器实测。

#### ⏱ 时序档位 + 地质级生命通道(world.html)
**档位**:速度按钮从「×倍数」改为**时间单位 8 档**,底层 `speedKind`+`speedRate`,两个引擎自动切换:
- **细档(小时/天/月/年)** `kind=min`:跑 `stepMinute`(生命+天气+扩散逐分钟),`rate`=每秒模拟分钟数(60/1440/43200/525600),`step()` 内封顶 24000 步/帧防卡。
- **粗档(10年/百年/千年/万年)** `kind=geo`:每帧跑 `rate` 个 `geoYear()`(封顶 400/帧)+ 各非洞穴地点 `geoLifeStep(r,GEO_SCALE)`。地质+生命同步快进。
**`geoLifeStep(r,scale)`(地质年生命步)**:复用现有 `fit`/资源/`updateLineages`,但用**弛豫积分**——`repl` 走 logistic 解析步 `K/(1+(K/N−1)e^(−r₀·scale))`(有利收敛承载K)或指数衰减(不利→灭绝),性状走指数弛豫 `+(1−e^(−a·scale))(opt−cur)`。**任意 scale 数值稳定不爆**,平衡 `repl≈38`(`GEO_CAP=40` 标定,对齐分钟级→保碳泵/O₂标定),性状(rTopt/rSalt/rDry/rUV/rAero/rEndo/rSymb/rAuto/rPhoto)朝当地最优演化。隔离验证:海洋起源→6物种分化、富氧 rAero→1、万年档跑 12000 地质年、确定性重跑一致。
**第一版范围**:粗档**不跑扩散**(扩散在分钟级 stepMinute),故粗档=地质演化+各地原地生命演化+谱系分化;跨地点迁移/生命登陆发生在细档。粗档下天气/水文(stepMinute)冻结(用进入粗档时的值),生命用冻结的 litAvg 等近似。碳泵/O₂ 的全局收支(GOE 结算)目前在 stepMinute 日结,粗档暂用 globalO2 现值——完整的粗档 GOE 结算可作后续。
- **海底热液喷口**：深海/近火山海区(`hvent`)喷热(局部暖点)+ 还原流体 **H₂/H₂S**(蛇纹石化, `redu`) + 金属硫化物烟囱(喂金属循环)。生命起源最主流的摇篮环境。
- **氧化还原 / 化学自由能梯度** `freeEnergy`：还原剂(`redu`，喷口)与海水氧化剂(CO₂/SO₄)**失衡 = 可用化学能**——第一代谢的能源。喷口处 ΔG 最高。
- **前生命"摇篮"就绪度** `cradleIndex`(0–1，显示用、不反馈)：双通道取 max——**喷口通道**(水×暖×自由能×硫化物催化×CHNOPS) / **地表通道**(水×温和×闪电能×干湿循环×黏土催化×CHNOPS)。验证：深海喷口~50%、湿地黏土池~23% 点亮；开阔洋/冻极/旱漠/洞穴≈0。两类经典起源环境自然涌现。
- **就绪清单**(全部非生物已满足)：液态水 ✓、还原性无氧 CO₂-N₂ 大气 ✓、闪电能(Miller-Urey) ✓、CHNOPS ✓、催化金属(Fe/Ni/Co/Mo/Zn/Mn/Se) ✓、矿物模板(黏土/硫化物) ✓、干湿循环(潮汐/蒸发湖) ✓、稳定气候+闭合养分循环 ✓、热液喷口+ΔG ✓。→ **模型已具备生命起源的非生物基础**；下一步"点燃生命"是生命层的事。

### 未接入
- **BodyModel**：人体存量-流量（体水/能量/电解质…），等"生命"层与之对接。
- **生命层（点燃生命）**：当 `cradleIndex` 够高的地点 + 足够地质时间 → 起源第一个自复制/代谢；接管碳/氮/氧循环（大氧化事件）、生物硅/有机碳埋藏、生物反硝化（前面特意留空的那些生物汇）。

## 7.5 适用域 / 边界（重要：别误用）

本模型从骨子里是一个 **「固体地表 + 薄流体包层（大气/海）」的类地天体模型**：地形/海拔、水循环、岩石循环、风化、沉积、板块、地点图、生存——**一切以基岩表面为参照**，能量回路以**入射太阳 vs 长波**为主导、内热为微小的放射成因热。

- **域内（能模拟，多为换配置）**：岩石行星、卫星、**水世界**（唯一陆地=火山岛，海底风化兜底碳恒温）。可调旋钮：行星参数(质量/半径/自转/轨道/金属丰度…)、纬度、世界类型、卫星/恒星列表(多卫星/双星)。
- **域外（撞穿根本假设，非调参可补）**：**气态/冰巨星（木星、海王星）、恒星**。原因：① **没有固体表面**——没有"地面"可站、"海平面"可参照，被模拟的那个地表世界整体失效；② **能量内热主导**（木星向外辐射 ≈1.7× 吸收阳光，来自原始/KH 收缩热），模型的主导项里没有；③ 环流是**深对流条带急流**而非类地薄壳 Hadley；④ 成分是 **H₂/He 包层**，无硅酸盐 → 无碳恒温器。要模拟它们需另起一个**竖直压力分层的流体包层模型**（压力坐标而非海拔坐标、内热驱动），那是另一类产品，不是本模型的开关。

> 验证旁注：把木星参数喂进**推导根**公式，`g=GM/R²`(25.9，实测24.8)、`H=RT/Mg`(H₂/He 23.8km，实测~27)、`Ω=2π/T`(2.4×地球)、`1/d²`(阳光仅1/27) **都给对**——推导层是天体无关的，边界出在它**下游**那个"地表世界"。

## 10. 比照真实生物进化进程：当前缺口与路线

把模型的演化弧对到真实地史的里程碑，标 ✓已有 / ◐部分 / ✗缺：

| 真实里程碑 | 模型 | 说明 |
|---|---|---|
| 生命起源（~3.8 Ga） | ✓ | cradleIndex 阈值确定性点燃 |
| 原核 / LUCA、化能自养 | ✓ | rAuto |
| 光合 → 放氧 | ◐ | 有产氧光合；**缺"不产氧光合"前驱阶段** |
| 大氧化 GOE（~2.4 Ga） | ✓ | 还原缓冲库→延迟跃升+BIF |
| 真核 / 内共生（~2 Ga） | ✓ | rEuk（线粒体；叶绿体未单列） |
| 有性生殖 / 减数分裂 | ◐ | rSex 标量；无真实重组/等位基因 |
| 多细胞（多次独立起源） | ✓ | rMulti |
| 雪球地球（~720 Ma） | ◐ | 有冰反照+CO₂机器；**未把"雪球解冻→动物兴起"接成因果** |
| 埃迪卡拉软躯体 | ✗ | 跳过软躯体阶段 |
| 寒武爆发 / 矿化骨骼 | ◐ | 有 rShell 外壳；**缺"门级体制爆发"本身（见下）** |
| 捕食 / 寄生军备竞赛 | ✓ | pred2 + parasite + 红皇后 |
| 神经 / 感官 | ◐ | rNeuro 当防御/觅食标量；非真正认知 |
| 三级营养金字塔 | ✓ | repl→cons→pred2 |
| 臭氧 + 登陆 | ✓ | rDry+rUV+臭氧门 |
| 陆生植物改造陆地 | ◐ | 根系加速风化抽CO₂；**缺维管/木质素/种子/真实森林** |
| 温血 / 体温调节 | ✓ | rEndo |
| 五大灭绝 | ◐ | 有失配灭绝脉冲；**未挂到具体成因（撞击/暗色岩省/缺氧）** |
| 个体实例化 | ◐ | creatureOf 只"描述"主导个体；**无真正离散个体/交互** |

**真实进化里、模型目前最缺的，按结构重要性排：**

1. **没有谱系 / 离散物种 / 系统树（最大结构缺口）**。模型是**平均场**：每地点的性状是连续标量、扩散时按生物量加权"混合"。因此**没有谱系分支、没有物种形成产生两个共存的姐妹种、没有支系（clade）级的灭绝与辐射、没有生物多样性计数、没有趋同/趋异作为离散实体**。真实进化是一棵**分叉的树**，这里是一片**滑动的性状均值场**。寒武爆发的本质是"**门级体制的爆发式分化**"，而模型只有一组标量在爬坡，表达不出"多样性爆炸"。

2. **没有基因型 / 没有变异之源 → 创新是预设菜单，不能开放式涌现**。"零随机"纪律下没有突变、没有中性漂变、没有标准遗传变异，适应=确定性梯度爬坡。代价是：**模型永远只能解锁设计者预先写好的那些 trait**，不可能"发明"一个没被预设的结构。真实进化的开放式新奇性部分来自随机突变 + 选择——这是"无随机"哲学换来的一个**根本性上限**（不是 bug，是取舍，但要明确）。

3. **没有体制 / 形态 / 发育（Bauplan）**。性状都是彼此独立的标量（壳、神经、体型…），**没有整合的身体计划**：对称性、体节、附肢、器官、发育约束、Hox 式模块。所以"组织分化"只是个标量，**没有真实的器官、体节、内骨骼/脊索→脊椎动物体轴、颌、鳍→四肢**。

4. **没有真正的离散个体与生活史**。仍是平均场的生物量池；creatureOf 只是**描述**一个代表个体，没有相互作用的离散生物种群、没有世代、没有卵/幼体/发育、亲代抚育。登陆的"繁殖脱水"（两栖→羊膜卵摆脱对水的依赖）也没建模（rDry 只管体细胞耐旱）。

5. **生态是线性链，不是网络**。有捕食-猎物、宿主-寄生张弛和三级金字塔，但**没有互利共生/协同进化**（传粉、菌根、地衣、肠道菌、珊瑚虫黄藻、固氮根瘤——模型只有寄生这一种种间关系）、**没有食物网网络**（只是直链 repl→cons→pred2）、没有生态演替、没有共灭绝级联。

6. **缺一批关键创新**：不产氧光合前驱、维管植物/木质素/种子（脱离对水的繁殖）、脊椎动物骨骼/颌/四肢/登陆四足、**飞行**（昆虫/翼龙/鸟/蝙蝠多次独立）、**被子植物–传粉者协同辐射**、社会性/真社会性、**认知/学习/文化/语言**（rNeuro 是防御标量，不是脑）。

7. **宏演化的"节奏与格局"缺**：纯渐变（标量连续爬坡）+ 阈值解锁，**没有间断平衡**（支系内长期停滞→突变）；灭绝是泛泛的 fit 触发，**没接到具体成因**（其实模型已有火山/CO₂/缺氧/冰期机器，可以把"大灭绝"挂上去做成**有名有姓的成因**；撞击体尚未建模）。

8. **几条具体的"地史–生命"耦合没闭合**：雪球解冻→动物兴起；泥盆纪陆生植物抽 CO₂→晚古生代大冰期（#8 抽了 CO₂ 但没把"→冰期→进化驱动"接成链）；石炭纪高 O₂→巨型节肢动物（呼吸气管限体型）+ O₂ 调控火灾频率。

**推荐的下一步（与"确定性/守恒/张弛"哲学相容、性价比高）：**
- **A. 把"大灭绝"接到既有成因**（火山暗色岩省脉冲 CO₂/酸化、冰期、缺氧）+ 加一个确定性撞击体（轨道交点式，像日月食那样无随机）→ 五大灭绝有名有姓。
- **B. 互利共生 trait**（与寄生对称的正项）：固氮根瘤/菌根/虫黄藻——补全"种间关系只有寄生"的缺口，且接 N 循环。
- **C. 把平均场升级为"少数离散支系"**：在一个地点维持 N 个 trait 簇而非 1 个均值，分异够大时**分裂成两簇=物种形成**，簇可独立灭绝——这是通向"多样性/系统树"的最小改动，也让寒武爆发真的"炸"出多样性。
- **D. 体制/形态层**：给个体加少量整合的形态变量（对称/体节/附肢/内骨骼），让 creatureOf 从"描述"走向真正的形态实例与交互。

> 一句话总结：**化学→细胞→代谢→大氧化→真核→多细胞→食物网→骨骼→神经→登陆→温血**这条**纵向**主干已经相当完整且全程确定性涌现；最大的不足是**横向的多样性**（谱系/物种/体制）和**开放式新奇性**（受"无随机"哲学限制），以及生态从"链"到"网"、灭绝从"泛泛"到"有名有姓"。
